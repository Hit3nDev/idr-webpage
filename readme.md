# Institute of Digital Risk (IDR) – Homepage

This project is a responsive homepage designed for the **Institute of Digital Risk (IDR)** as part of a design and development assignment.

## Project Overview

The website introduces IDR as an industry-led institute focused on training and deploying professionals in digital, cyber, and AI risk.  
The site was built using **HTML, CSS, and JavaScript** with a responsive layout and modern design.

## Logo Design Rationale

The icon of the cube was selected as it symbolizes structure, stability, and interrelated systems, which is in line with the analysis and management of digital risk.  
Orange was selected as the accent color as it symbolizes urgency and attention, whereas black adds a professional touch.  
White was selected as it provides maximum contrast with varying screen sizes.  
The Syne typeface was selected as it is modern and geometric in appearance, thus giving the brand a professional look.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
